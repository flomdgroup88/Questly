# Настройка Firebase для Questly (5 минут)

Firebase нужен, чтобы соревнования хранились в облаке и друзья
могли находить их по коду. Бесплатный тариф (Spark) полностью достаточен.

---

## Шаг 1 — Создать проект Firebase

1. Открой https://console.firebase.google.com
2. Нажми **"Добавить проект"**
3. Введи название: `questly-social`
4. Отключи Google Analytics (не нужен) → нажми **"Создать проект"**

---

## Шаг 2 — Создать базу данных Firestore

1. В левом меню выбери **Firestore Database**
2. Нажми **"Создать базу данных"**
3. Выбери режим **"Тестовый режим"** (разрешает запись на 30 дней)
4. Выбери регион: `europe-west` → нажми **"Готово"**

> После 30 дней нужно будет обновить правила. Напиши мне — помогу.

---

## Шаг 3 — Получить конфиг приложения

1. На главной странице проекта нажми на иконку `</>` (Web)
2. Придумай имя приложения: `questly-web` → нажми **"Зарегистрировать"**
3. Скопируй объект `firebaseConfig` — он выглядит так:

```js
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "questly-social-xxxxx.firebaseapp.com",
  projectId: "questly-social-xxxxx",
  storageBucket: "questly-social-xxxxx.appspot.com",
  messagingSenderId: "1234567890",
  appId: "1:1234567890:web:abcdef123456"
};
```

---

## Шаг 4 — Вставить конфиг в проект

Открой файл `src/firebase.js` и замени блок `firebaseConfig`
своими данными из шага 3.

---

## Шаг 5 — Установить зависимости и запустить

```bash
npm install
npm run dev   # для разработки
npm run build # для деплоя
```

---

## Как это работает теперь

- **Создать соревнование** → данные сохраняются в Firebase по `shareCode`
- **Поделиться кодом** → друг вводит код, приложение ищет в Firebase
- **Присоединиться** → соревнование добавляется к другу локально,
  его имя добавляется к участникам в облаке
